# Shop System — 商店系统

## 局内商店

### 货架规则

- 货架数量：4 格。
- 初始叶铢：200（每关开局重置）。
- 刷新花费：5 叶铢。

### 生成规则

| 货架位置 | 内容 |
|---------|------|
| 第 1 格 | 必定为转向器（随机方向） |
| 第 2\3 格 | 50% 武器 / 50% 转向器 |

- 武器池：当前卡组已选择的 5 款武器。
- 转向器价格：5 叶铢（固定），货架展示具体方向。

### 武器价格

| 武器 | 价格(叶铢) |
|------|:----------:|
| pistol | 10 |
| bomb | 16 |
| lightning | 18 |
| laser | 14 |


### 回收规则

```
返还叶铢 = floor(基础价格 × 等级 / 2)
```

- 可回收物品：武器、转向器。
- 拖拽物品到回收区上方时，回收区高亮提示，并显示预估的退款金额。
- 松开后获得对应叶铢，物品从棋盘上移除。

### 购买规则

- 拖拽商店物品到空格 → 购买并放置。
- 拖拽到同类同级物品 → 购买并合并（等级+1，不额外消耗）。
- 叶铢不足时不扣费，提示余额不足。
- 目标格被不可合并物占据时提示。
